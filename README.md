# Task Scheduling Environment (OpenEnv)

This project implements a simple task scheduling environment using OpenEnv.

## Features
- Custom environment with tasks (Study, Exercise, Sleep)
- step(), reset(), and state() functions
- Reward-based scheduling system
- Easy, Medium, Hard graders

## Files
- env.py → Environment logic
- agent.py → Agent actions
- grader.py → Evaluation
- test_env.py → Testing
- openenv.yaml → Config file

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run test:
   python test_env.py

3. Run grader:
   python grader.py

## Output
- Shows task scheduling steps
- Displays reward
- Evaluation scores (easy, medium, hard)# Task Scheduling Environment (OpenEnv)
